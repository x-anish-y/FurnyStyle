# 🪑 FurnyStyle – Modern Furniture E-Commerce Platform

FurnyStyle is a modern furniture browsing and shopping platform designed to showcase stylish furniture collections with a clean, responsive user interface.

The project demonstrates modern frontend design principles and scalable web architecture using Next.js and React.

---

## 📌 Project Overview

FurnyStyle provides users with a visually engaging platform to explore furniture products, discover interior design inspiration, and browse curated collections.

The application focuses on:

- Elegant product presentation
- Responsive design across devices
- Smooth navigation and UI interactions
- Scalable frontend architecture

---

## 🚀 Core Features

- 🛋️ Modern furniture product showcase
- 🔍 Product browsing and category navigation
- 📱 Fully responsive design
- ⚡ Fast page loading with Next.js
- 🎨 Clean UI with modern styling
- 🧩 Reusable component-based architecture

## ⚙️ Application Workflow

## 🔧 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/x-anish-y/FurnyStyle.git
cd FurnyStyle
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
